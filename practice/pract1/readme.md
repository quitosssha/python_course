Практика1.

1. Дано количество имён N. Вводится N имен и помещаются в список. Создайте новый список uni:
- Пройдите по исходному списку names и добавьте имя в uni только в том случае, если имени с такой длиной в uni ещё нет. В итоге в uni должны быть имена разной длины.
- Выведите оба списка.

2. Дано число N. Создайте список чисел от N до N^2 с помощью range(). Используя генератор списка, создайте новый список, содержащий квадратные корни этих чисел. Выведите оба списка.


3. Через пробел вводятся числа. В следующей строке вводится число K. Используя генератор списка, создайте из введенных чисел список всех чисел, кратных K. Выведите результат.


4. В одну строку вводятся фразы, разделенные символом “;”. Используя генератор списка, создайте список lengths, содержащий количество слов в каждой фразе из исходного списка. Выведите исходный список и список lengths.


5. В строку вводится список цен на товары в магазине. В следующих строках вводятся число K и число M.
В магазине действует акция: каждые полные К рублей в цене товара дают скидку М рублей.
Используя один генератор списка, создайте новый список discount, который будет содержать цены после применения скидки. Выведите списки цен до скидки и после применения скидки.
